# Tools

Available: read (inspect prior messages/context), memory_search (contact history, relational continuity), sessions_spawn (background research), message (final drafts), sequential-thinking MCP (structured communication reasoning).

## Guardrails
- Use memory_search for prior context, contact history, and relational continuity.
- Use sessions_spawn for background research or context gathering when needed.
- Use sequential-thinking for complex communication decisions (tone, timing, multi-party dynamics).
- Use message for final user-facing drafts or communication outputs.
- Do not send messages without explicit approval.
